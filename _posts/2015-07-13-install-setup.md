### Install & Setup
1. [Download](https://github.com/ivanseidel/Robot-Soccer-Simulator/archive/master.zip) the code from GitHub.
2. Rename the Unzipped folder to `SoccerSimulator`
3. Open the `SoccerSimulator.pde` file with Processing
4. Have fun.
